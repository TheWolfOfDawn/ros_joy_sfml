# ros_joy_sfml

## Description :
A simple ROS tool to move a robot with a joystick, it has been tested with an Xbox controller and a PS-5 controller

## Required :
SFML
```sudo apt-get install libsfml-dev```

## How to start :
```rosrun driver_manette driver_manette```
